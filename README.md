## About This Project
The **Travel Agency Page** expands on previous projects by introducing more semantic structure and real-world layout patterns.

This project simulates a travel website where users can explore destinations, view packages, and browse featured itineraries.

Key concepts explored:

- Proper HTML document structure (`DOCTYPE`, `html`, `head`, `body`)
- Using meta tags for SEO and page information
- Organizing content with headings (`h1`, `h2`)
- Creating navigation links with anchor (`a`) elements
- Structuring grouped content using lists and figures
- Embedding images with captions using `figure` and `figcaption`

---

## Key Features
- Travel-themed heading and introduction section  
- “Packages” section with:
  - Group Travels  
  - Private Tours  
- “Top Itineraries” section featuring:
  - Clickable images  
  - Captions for each destination  
- External links that open in a new tab
